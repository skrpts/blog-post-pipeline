## Purpose

Reference document for the SEO optimisation skill. Defines the standards against which content is evaluated for search engine visibility.

## On-Page SEO Standards

### Keyword Placement

- Target keyword must appear in: page title, H1, first 100 words, at least one H2, meta description
- Secondary keywords should appear naturally in H2/H3 headings and body text
- Keyword density target: 1-2% for the primary keyword
- Avoid keyword stuffing — readability always takes priority

### Heading Structure

- Every page needs exactly one H1 that includes the target keyword
- Use H2 headings every 200-300 words to break up content
- H2 and H3 headings should be descriptive and, where natural, include related keywords
- Use question-format headings where they match common search queries

### Content Length Guidelines

| Content Type | Minimum Words | Target Words |
|-------------|---------------|--------------|
| Blog post | 800 | 1,200-2,000 |
| Pillar page | 2,000 | 3,000-5,000 |
| Landing page | 300 | 500-800 |
| Product page | 200 | 400-600 |

### Internal Linking

- Every piece should link to at least 2-3 related internal pages
- Use descriptive anchor text (not "click here" or "read more")
- Link to pillar content from supporting articles
- Update older content to link to new related pieces

### Meta Data

- **Meta title:** under 60 characters, includes target keyword, compelling
- **Meta description:** under 155 characters, includes target keyword, contains a call to action
- **URL slug:** short, descriptive, includes target keyword, uses hyphens

### Image Optimisation

- All images must have descriptive alt text
- Include target keyword in at least one image alt tag
- Use descriptive file names (not IMG_001.jpg)
- Compress images to under 200 KB where possible
