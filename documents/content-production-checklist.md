## Purpose

A working checklist to track progress through the blog post production pipeline. Use this as a quality gate before publishing.

## Pre-Writing

- [ ] Topic approved and assigned
- [ ] Content brief completed and reviewed
- [ ] Target keyword(s) confirmed with search volume data
- [ ] Competitor content analysed (top 3 ranking pieces)
- [ ] Reference material gathered

## Writing

- [ ] Draft follows the content brief outline
- [ ] Target word count met
- [ ] Primary keyword included in title, first paragraph, and at least one H2
- [ ] Internal links added (minimum 2-3)
- [ ] Images selected with descriptive alt text
- [ ] Call to action included

## Editorial Review

- [ ] Grammar and spelling checked (British English)
- [ ] Style guide compliance verified
- [ ] Brand voice alignment confirmed
- [ ] Consistency check passed (terminology, formatting)
- [ ] All facts and statistics verified with sources

## SEO Review

- [ ] Meta title written (under 60 characters)
- [ ] Meta description written (under 155 characters)
- [ ] URL slug optimised
- [ ] Heading structure validated (H1 > H2 > H3)
- [ ] Keyword density within 1-2% range
- [ ] Image alt tags completed

## Publishing

- [ ] Final proofread completed
- [ ] CMS formatting verified (headings, lists, images render correctly)
- [ ] Social sharing metadata configured (Open Graph, Twitter Card)
- [ ] Publication date and time confirmed
- [ ] Distribution channels notified (newsletter, social, team)
